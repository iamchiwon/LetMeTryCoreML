# 요즘 유행하는 AI 나도 해보자 (feat. CoreML)

[Let's Swift 2023](https://letswift.kr/2023) 발표 자료

### Why CoreML?

- 향상된 사용자 경험 제공
- 실시간으로 대응되는 빠른 반응
- 개인 데이터의 외부 유출로부터 안전
- 오프라인에서도 동작
- 모바일 장치들의 성능이 향상
